Sokoban est un jeu créé en langage C. Il se joue sur un terminal. C'est un jeu dans lequel un personnage doit pousser une boite jusqu'à un emplacement précis. 
Le personnage est symbolisé par un rond "o" et la boîte par une croix "X". 
Si la boîte arrive au bon emplacement symbolisé par un point "." je joueur a gagné. 
Si la boite ne peut plus bouger (par exemple si elle est collée à un mur ou dans un coin) la partie est perdue.
Le personnage peut pousser une caisse, mais jamais la tirer.
